## Platform Overview

Shiftza is a full end-to-end developer intelligence platform. Here's everything it does:

---

### 🔍 AI-Powered Search  
`/search`

The core of Shiftza. Describe what you need in plain English — no query syntax, no booleans.

- **Natural language input** — write like you're explaining the role to a colleague
- **AI query expansion** — Shiftza breaks your description into domain-specific search vectors automatically
- **Real-time progress** — watch Shiftza scan and rank profiles live with an animated agent loader
- **Smart suggestions** — niche-specific search prompts to help you get started instantly
- **Search history** — every search is saved; resume or re-run any past query in one click
- **Discuss & Refine** — after results load, continue a conversation to narrow down further
- **Upgrade prompts** — free users see a usage counter and upgrade flow when limits are reached

```
Example query:
"Senior backend engineer with Kafka, microservices, 
and fault-tolerant distributed systems in production"
```

---

### 📋 Search Results  
`/search/[id]`

Every search produces a persistent, shareable results page.

- **Ranked developer cards** — ordered by evidence strength, not guesswork
- **Domain match score** — how well each engineer fits your exact query
- **Quick-view panel** — click any developer to expand a full profile without leaving results
- **Shortlist action** — save any developer to your bookmarks instantly from results
- **Outreach button** — generate a personalised email for any developer directly from the results list
- **Filter sidebar** — narrow results by location, seniority, activity, and tech keywords
- **Funnel counter** — see how many profiles match at each filter stage in real time

---

### 👤 Developer Profiles  
`/[github-username]`

Every developer on GitHub has a clean, AI-enriched profile on Shiftza — no sign-up needed to view.

**Access it in two ways:**

```
# Option 1 — Direct URL
shiftza.in/torvalds

# Option 2 — The Shift trick
# On any GitHub profile, just prepend "shift" to the URL:

github.com/torvalds
     ↓
shiftgithub.com/torvalds  →  automatically redirects to shiftza.in/torvalds
```

**What's on each profile:**
- Real name, avatar, bio, location, company, and social links
- Top programming languages (by actual code, not self-report)
- Top repositories — stars, forks, description, primary language
- Recent contribution activity
- GitHub followers & following
- AI-generated summary of what they actually build
- Bookmark button — save directly to your watchlist
- Outreach generator — write a personalised email referencing their real work

---

### 🔖 Saved Developers  
`/watchlist`

Your personal developer pipeline — bookmark and organise engineers you want to track.

- **Save from anywhere** — bookmark from search results, the filter sidebar, or a profile page
- **Multiple watchlists** — create named lists to organise by role, project, or team
- **Per-developer notes** — attach a private note to any saved developer
- **Saved search context** — see which query surfaced each saved developer
- **Relative timestamps** — "Saved 2h ago", "Saved 3d ago"
- **One-click to profile** — jump straight to any saved developer's full profile
- **Remove from list** — clean up your pipeline with a confirmation dialog

---

### 🕐 Search History  
`/history`

Every search you've ever run is stored and retrievable.

- Full list of past queries with display title and result count
- Relative timestamps — "Just now", "3h ago", "5d ago"
- Click any item to jump straight back into those results
- Auto-updates when new searches are run

---

### ✉️ AI Outreach Generator

Available from search results and developer profiles.

Generates a personalised cold outreach email that:
- References the developer's **actual repositories** and **specific contributions**
- Mentions their **top languages and domain expertise** pulled from real GitHub data
- Reads like it was written by a human who did their homework — not a template
- Is editable before you send

Free plan: **3 emails/day** · Team plan: **Unlimited**

---

### ⚙️ Settings  
`/settings`

Your account dashboard.

- **Account overview** — email, auth provider, member since, last sign-in
- **Usage meter** — live search count vs. your plan limit with a progress bar
- **Subscription details** — current plan (Free / Team), billing type, status
- **Upgrade path** — direct link to book a demo and talk to the team
- **Security** — sign out from the current device
- **Support** — direct email link to the Shiftza support team

---

### 🔐 Authentication  
`/auth`

Full auth flow — no GitHub dependency required to use the platform.

- Sign up with **email + password** or **OAuth** (GitHub / Google)
- Email verification flow
- Forgot password & reset password
- Secure session management

---

### 🌗 Dark / Light Mode

System-wide theme toggle available on every page. Defaults to dark.
